Chart 1 — Vector Space

Notes for Chart 1 — Vector Space

Observation: The reference chart shows a two-dimensional vector space with four labeled basis vectors and directional arrows.

Implementation: I used D3.js to draw an SVG grid and manually placed four basis points. Dashed lines with arrow markers were created using line elements. Each line animates on load and thickens on hover to fulfill the interaction requirement.

Reflection: This chart helped me understand how D3 manages coordinates and vector direction visually. Smooth transitions make relationships clearer and more dynamic.

Chart 2 — Word Network

Notes for Chart 2 — Word Network

Observation: The reference diagram connects semantically related words into two clusters, forming a visual network.

Implementation: Each word is manually positioned with x/y coordinates and connected by dashed lines. Hovering over edges thickens them, and all labels fade in on load.

Reflection: This visualization shows how D3 can organize semantic data spatially. Even minimal transitions improve clarity and engagement.

Chart 3 — Optimization Route

Notes for Chart 3 — Optimization Route

Observation: This chart visualizes a gradual optimization path moving toward a target point.

Implementation: I drew a dashed polyline using d3.line() and animated it using stroke-dash transitions, creating a self-drawing route.

Reflection: This helped me practice D3 path animation. The progressive reveal effectively communicates direction and iteration in optimization processes.

Chart 4 — Cluster Scatter

Notes for Chart 4 — Cluster Scatter

Observation: The reference scatter plot depicts two distinct data clusters to illustrate grouping or similarity.

Implementation: I generated two random clusters with slight jitter. Each point drops in from above while fading in. Hovering enlarges the circles to highlight interactivity.

Reflection: This chart shows how motion adds liveliness to static scatter plots. The transitions make the clustering pattern immediately perceptible.

Chart 5 — Mixed Math Plots

Notes for Chart 5 — Mixed Math Plots

Observation: The chart combines discrete and continuous mathematical visualizations: small bars on the left and continuous curves on the right.

Implementation: Bars represent discrete steps with Δn shift. Two right-side curves are drawn as solid and dashed lines in a green scheme, animated progressively using stroke-dash offset transitions.

Reflection: This exercise showed how D3 can represent both discrete and continuous functions in one SVG canvas. Gradual animation adds rhythm and continuity to mathematical visualization.
